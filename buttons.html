Glowing Button

This code creates a stylish animated button component. The button has a gradient background that transitions through various vibrant colors, creating a dynamic and eye-catching effect. The text inside the button is displayed in a bold typeface called "Space Grotesk", adding to the modern aesthetic. Additionally, there's a subtle hover effect that enhances the animation. The component is designed to be responsive, adjusting its size for smaller screens. Overall, it's a visually appealing and interactive element that can be used to enhance the user experience on websites or applications.
